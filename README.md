# 简单链接收集器

一个超简单的 Web 应用，允许用户通过网页提交和查看链接（例如书签或资源列表）。本项目使用 Python 和 Flask 构建，适合初学者学习 Web 开发。

## 功能
- 通过网页添加链接。
- 显示所有已保存的链接。
- 简单易用的代码库，欢迎开源贡献。
